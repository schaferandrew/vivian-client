# Vivian Client

Next.js PWA for the Vivian household assistant.

## Development

```bash
npm run dev      # dev server
npm run build    # production build
npm run test     # run tests
npm run lint     # eslint
```

## Permissions

```json
{
  "permissions": {
    "allow": [
      "Bash(gh issue create:*)",
      "Bash(gh issue list:*)",
      "Bash(gh pr create:*)"
    ]
  }
}
```
